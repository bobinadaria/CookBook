# scripts/archive

Исторические one-off скрипты и отчёты — уже отработали свою задачу, оставлены для справки.
Не запускать как часть обычного пайплайна; актуальные скрипты — в `scripts/`.

- `add-*.mjs`, `finalize-*.mjs`, `translate-and-gen.mjs` — разовый сидинг старых рецептов.
- `analyze-all-recipes.mjs`, `dump-*.mjs`, `fill-cook-time-servings.mjs` — разовые аудиты/бэкфилы.
- `migration-cook-time-servings.sql` — применённая миграция.
- `ingredients-audit-report.md` — снимок аудита `ingredients_base`.
