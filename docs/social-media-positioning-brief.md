# The Slow Table — позиционирование для социальных сетей

Краткий бриф для переиспользования при написании постов, карусели, Threads, LinkedIn. Это быстрый справочник — полное тактическое руководство в `docs/smm-content-guide.md`.

---

## Продукт в одном предложении

Личная кулинарная книга с AI-нутрициологом — место где хранятся рецепты и истории к ним, не папка с закладками.

---

## Три кита позиционирования

**1. Воспоминание, а не контент**
Контент можно сгенерировать, а воспоминание нет. Рецепты в The Slow Table — это не просто список ингредиентов, это папин плов, второе свидание, мамин торт. Это то, что ChatGPT не придумает.

**2. Женщина с несколькими источниками дохода**
Дарья — соло-фаундер, запустила продукт без команды и инвесторов. Этот угол работает для аудитории, которую вдохновляют женщины с финансовой независимостью. Уязвимость ("мне казалось это недостижимым") + результат = сильный крючок.

**3. AI честный, а не халтурный**
КБЖУ считается по базе данных Минсельхоза США, не генерируется из воздуха. Это дифференциатор от ChatGPT и "хаотичного генерирования цифр". Доверие через конкретику.

## Структура карусели (Hook / Story / Offer)

Слайд 1 — хук. Останавливает скролл. Личный факт + неожиданный поворот + интрига. Обязателен призыв листать.

Слайд 2 — проблема читателя. Называет боль конкретно. Читатель должен узнать себя ("в переписке с мамой трёхлетней давности"). Не продающий — строит доверие.

Слайд 3 — личная история. Самый эмоциональный. Конкретная деталь (казан, плов, папа) важнее красивого текста. Заканчивается пивотом на читателя ("у тебя тоже есть такие рецепты").

Слайд 4 — фичи через действие. Не список, а сценарий: "кидаешь ссылку → появляется в книге". Нейтральный по полу (не "нашла", а "понравился рецепт").

Слайд 5 — CTA. Конкретное действие + адрес. Никаких расплывчатых "узнай больше".

## Тон of voice

**Что работает**
- Короткие предложения, иногда фрагменты с точкой. "Каждый. Раз."
- Конкретные детали вместо общих слов. Не "дорогое место", а "зирвак булькает, звоню папе".
- Уязвимость без жалости. "Мне казалось это недостижимым" — это сила, не слабость.
- Прямое "ты" к читателю.
- Риторический вопрос + немедленный ответ. "А рецепт где? В скриншотах."

**Что убирать**
- Тире (воспринимаются как сгенерированный текст)
- Корпоративный язык: "реализация", "диджитал формате", "инструмент"
- Слова-паразиты: "при этом", "стоит отметить", "без особого таланта"
- Пассивные конструкции: "можно будет использовать", "является"
- Уменьшение себя: "я просто", "без таланта", "немного"
- "Скоро" для фич, которые уже работают

## Сильные фразы для переиспользования

"Контент можно сгенерировать, а воспоминание нет."
"В скриншотах/сохранёнках/в переписке с мамой трёхлетней давности."
"Место где чувствуется жизнь."
"Никакого хаотичного генерирования цифр из ChatGPT."
"Выглядит как настоящая книга."
"Только для тебя. Никто больше не увидит."

## Углы для разных платформ

**Instagram карусель** — личная история + фичи + CTA. Формат как у этой карусели.

**Threads** — один слайд карусели = один тред. "Контент можно сгенерировать, а воспоминание нет" — готовый тред-хук. Или "три источника дохода в 31 год" + история в треде.

**LinkedIn** — акцент на соло-фаундер угол. Техстэк, без команды, без инвесторов. Финансовая независимость как профессиональный выбор. Менее личное, больше про процесс и решения.

**Для любой платформы** — первые две строки решают всё. Дальше текста читатель не видит без клика.

## Аудитория

Основная: женщины 25-40, интересуются едой, путешествиями, финансовой независимостью. Не обязательно IT.
Вторичная: IT-аудитория, которую вдохновляет соло-запуск продукта.
Не аудитория: B2B, рестораны, профессиональные повара.
